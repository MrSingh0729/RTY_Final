About FPY Dashboard

The FPY Dashboard is an advanced analytics platform designed to monitor and analyze First Pass Yield (FPY) metrics in manufacturing environments. It provides real-time insights into production performance, helping teams identify issues, track improvements, and make data-driven decisions.

  • Real-time data monitoring
  • Advanced filtering and search
  • Comprehensive failure analysis
  • Export capabilities (Excel, PDF)
  • Mobile-responsive design
Data Sources

The dashboard integrates with multiple data sources to provide comprehensive insights:

Data Source Description
MES Factory Report API Real-time production data
SQLite Database Model descriptions and goals
Project Management System Active project list

Key Metrics Tracked

RTY (Rolling Throughput Yield)

Overall production yield from start to finish

NG (No Good) Rate

Percentage of products failing quality checks

NDF (No Defect Found) Rate

Products flagged as defective but found OK on recheck

Station Performance

Detailed analysis by production station
